PORTFOLIO



Book Master

- A frontend-only React app that maintains an in-memory database of books. The user, through the simple interface shown below, can look up and add books from the Open Library API, view the list of books and edit certain details of the book.
1. View all books in the database
a. List the title, publishor, author and a short description of the book
b. The book list should be sortable by the order added and by title, in ascending and descending order
c. Allow the user to select a book to edit
2. Add a new book
a. Allow the user to enter an Open Library ID to add a book
b. Newly added books appear in the list, in appropriate order
3. Edit an existing book
a. Allow the user to select a book to edit
b. Display current field values and allow the user to edit
